# clinic_app
A healthcare-focused web application designed to streamline clinic workflows. Features include a patient booking portal, clinical notes for physicians, and an administrative dashboard for manual scheduling.

Tech stack _ Frontend: TML5, CSS3, JavaScript _ Backend: firebase ( firestore and authentication)

Key Feature patient booking portal: Allows patients to easily schedule appointments digitally. provider baseboard: A centralized view for doctors to monitor their daily schedule and patient flow. clinical notes: A dedicated space for physicians to document patient consultations and important health notes in real-time. Administrative Overrides: Built-in functionality for clinic secretaries to manually add or adjust appointments, ensuring flexibility for walk-in patients or phone bookings.
